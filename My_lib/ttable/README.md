### 使用

```
    x = ttable("Test title",4)
    x.header("column1","column2","column3","column4")
    x.append("t1","t2","very long entry","test")
    x.append(("r","r3"),("l","l2"),"also long entry","test")
    print x
```

#### 表格中内容颜色

```
("txt","3"),其中的数字则为颜色
```
#### 表格中对齐方式

* l 靠左
* r 靠右
* c 居中
```
("txt","l")
```
